Audio Interface

Includes:
"CommonServices.h"

Introduction

Provides APis related to Audio rendering on the accessory.


Groups

VolumeAdjuster

API for adjusting volume to minimize glitches.

Group members:

AudioStreamCreate

Creates a new AudioStream.

AudioStreamGetLatency

Gets the minimum latency the system can achieve (may be higher if samples are already queued).

AudioStreamGetVolume

Gets the current volume of the stream as a linear 0.0-1.0 volume.

AudioStreamPrepare

Prepares the audio stream so things like latency can be reported, but doesn't start playing audio.

AudioStreamSetAudioCallback

Sets a function to be called for audio input or output (depending on the direction of the stream).

AudioStreamSetFlags

Streams the AudioStream.

inFlags will be kAudioStreamFlag_Output.

AudioStreamSetFormat

Sets the format to provided to the callback for input or the format provided by the callback for output.

AudioStreamSetPreferredLatency

Sets the lowest latency the caller thinks it will need. Defaults to 100 ms.

AudioStreamSetThreadName

Sets the name of threads created by the clock.

AudioStreamSetThreadPriority

Sets the priority of threads created by the clock.

AudioStreamSetVolume

Sets the volume of the stream to a linear 0.0-1.0 volume.

AudioStreamStart

Starts the stream (callbacks will start getting invoked after this).

AudioStreamStop

Stops the stream.
